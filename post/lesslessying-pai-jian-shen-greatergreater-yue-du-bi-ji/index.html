《硬派健身》阅读笔记

2021-04-11 6 min read

Intro：当我们在聊健身的时候，我们在聊些什么？

1.变得更好

  • 环境：人们免不了肤浅、势利、刻薄。尤其是今天，互联网放大了一切细枝末节，所有这一切就像是被放在显微镜下，被公众讨论、嘲笑、奚落。
  • 所以，每个人都希望自己变得更好、看起来更健康，而不仅仅是快乐。

2.健身不是目的，自己才是目的

  • 健身的目的：身材不是你健身的目的，快乐也不是健身的目的，甚至健身都不是你的目的。你的目的，是发现你自己，困缚于日常生活中的自己。
  • 与身体的关系
    • 初始，我跟我的身体对抗。目的性很强的健身效果很差，不仅身材没有变好，运动对我来说，都变成了一种压力。
    • 过程中，我和身体和解。发现自己身体每一个部位存在的意义，我可以渐渐地了解它们、熟悉它们。健身是自己和自己的身体一起，为了更好的生活去努力。

3.尊重身体

  • 尊重自己的身体，尊重科学，自己去寻找和发现健身中的各种知识和宝藏。

C1：什么样的身材是好身材？

  • 好身材：富有生命力的身材，瘦而有型。“型”是骨骼和肌肉的形态与搭配，骨骼可能是天生的，肌肉却不是。

1.一切从大肌群训练开始

  • 人体的肌肉是由大肌群（胸、背、臀、腿）和小肌群（肩、胳膊、腹、小腿）构成的。人体的骨骼就像一幢房子的基本结构，大肌群就像墙体，保护内脏、填充身体；小肌群更像是窗帘、茶几之类的小装饰，某种程度上更像一种整体风格的延伸。
  • 为什么最应该练胸、背、臀、腿的大肌群？
    • 大肌群与体形的关系：大肌群修饰整体体形，是第一眼就能看到的部位，可以塑造一种直观上整体完美的体形。小肌群是只有自己才在乎的微小细节。
    • 减脂增肌角度：从减脂角度看，相对于大肌群，小肌群消耗的热量是微不足道的，在减脂塑身方面效果甚小；对于增肌者来说，也只有大重量、大肌群的训练才能更好地刺激各种激素（睾酮或生长激素等）的分泌。

给训练入门者一个诚心诚意的忠告：放弃大多数小肌群训练，从今天起，每天做一次大肌群训练（胸、背、臀、腿）6～10组，搭配核心肌群训练（腹部、下背部）1～3组，最后做HIIT（高强度间歇训练）15到30分钟。不到半年，你就会惊喜地看到自己的改变。

2.体型和体脂

  • 体重不是关键。相同身高体重但不同体脂，体型差异是巨大的。
  • 肌肉的好处：
    • 密度更大，相同重量下体积更小。
    • 肌肉组织的能量消耗更大。

C3：什么样的有氧运动最减脂？

1.长时间的有氧运动不是减肥的最好方法

  • 消耗脂肪的同时也在消耗肌肉，直接影响人体的新陈代谢效率。
  • 对体内内分泌起到的作用类似于节食，导致体内瘦素减少，实际上是促使变胖。

2.HIIT才是普通人减肥塑身的最好选择

  • 定义：HIIT（High-intensity Interval Training，高强度间歇训练）。指多次短时间、高强度的运动训练，在每两次高强度运动之间以较低强度的运动或完全休息形成间歇期。简单地说，HIIT就是一种高强度运动与低强度运动间歇交替的训练方式，
  • 特点在于：（1）短时间内运动强度较大，每次都要达到最大或至少接近最大的运动能力。（2）运动时间相对较短，并且可以通过间歇期间的低强度，来避免不适症状的出现。
  • 优点：
    • （1）高强度：持久减脂，刺激生长激素分泌；
    • （2）间歇：强效减脂，促进脂肪代谢
      • 有间歇的运动中，脂肪参与供能的比例更大，燃烧的脂肪也更多。
  • HIIT在某种程度上是无氧肌肉训练和有氧训练的结合，不仅有助于减去总脂肪量、体脂百分比，而且提升了安静时脂肪的氧化率以及总体的新陈代谢。
  • 高强度间歇运动使受试者获得的愉快感要强于中等强度的持续性运动。

3.如何设计HIIT

  • 先让身体习惯动起来，再逐步加强。不要给自己设置太多的阻力，或者太多虚幻的梦想。只要你能稳住心态，正确且坚持锻炼，好身材迟早会有的。
  • 肌群参与得越多，燃脂效果就越好。HIIT的一个不成文的特点就是让身体尽可能多地活动起来。

3.1. 减脂HIIT：爬楼梯

  • 爬楼梯的燃脂效果好：运动到的股四头肌和臀大肌都是燃脂能力高且塑形能力好的大肌群。爬楼梯在运动方式上，实际非常接近短间歇的无氧训练，所以每次爬楼，还可以带来48到72个小时的持续燃脂。
  • 爬楼梯伤膝盖？
    • 真正伤害到膝盖的并非“爬楼梯”，而是“下楼梯”。爬楼梯上去，再乘坐电梯下来。
    • 掌握正确的姿势：使用臀大肌发力&保证重心靠后
      1. 一次上两个台阶（大腿和上半身之间的髋角更大，对臀部的刺激更大）
      2. 背部挺直
      3. 膝盖不能超过脚尖
      4. 念动合一：心中默念“用臀部发力，用臀部发力，用臀部发力……”
  • HIIT形式，交替三种速度（快速冲刺、较快、一般）爬100层。每周三次，每次半小时。

3.2. 塑形HIIT

  • 【跳跃拍手+直膝前踢+俯卧登山】每个动作一分钟，休息10秒，为一组。每次4组，每周3次。
